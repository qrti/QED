Short description:
QED is a full featured, fast and rock stable working editor for large text documents on PalmOS organizers. It is FREE usable as reader! Ideal for reading and writing all kind of documents including lists and source code. Intuitive and unique user interface, practical oriented operation, fast compression, wide virtual screen, categories, bookmarks, privacy. Breaks the memopad limit and can edit standard compressed DOC files larger than physical memory. Prior updates included access to all fonts, keyboard support, relative screen tap scrolling, search/replace with options, global find, Save As, BackupBuddy certified, comprehensive step by step manual, German, Japanese, Spanish FAQs. -> more

QEX2 is an easy to use Windows program to convert text documents between PC and PalmPilot format (both directions). The program supports Windows 95, 98, NT, 2000 and XP. QEX2 is Shareware, unregistered QEX2 is fully functionable but registration requesters will appear on program start and during work. -> more

QEX is an easy to use Windows program to convert text documents between PC and PalmPilot format (both directions). The program supports Windows 95, 98, NT and 2000. Registered users of QED can use their registration key for QEX as well, unregistered QEX is fully functionable but registration requesters will appear when starting it. -> more
